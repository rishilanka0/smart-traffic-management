# Smart Traffic Management AI 🚦

An AI-based traffic management system that uses YOLO for real-time vehicle detection and reinforcement learning to optimize traffic signal timings.

## 🚀 Features
- Real-time vehicle detection using YOLO
- Adaptive traffic signal control using RL
- Emergency vehicle (ambulance) prioritization
- Reduced congestion and improved traffic flow

## 🛠️ Tech Stack
- Python
- YOLOv8
- Reinforcement Learning
- OpenCV

Run the main file to start the system:
